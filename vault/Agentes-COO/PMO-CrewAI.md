# PMO-CrewAI

**Qué hace:** Explora la intersección entre IA y gestión de proyectos usando agentes autónomos de CrewAI dentro de un entorno de Project Management Office.

**Repositorio:** [github.com/ai-in-pm/PMO-CrewAI](https://github.com/ai-in-pm/PMO-CrewAI)
**Relacionado:** [PMO-AI-Agents](https://github.com/ai-in-pm/PMO-AI-Agents) — versión con estructura jerárquica más elaborada (Supreme Oversight Board, Executive Agents, Sub-Agents), mismo autor, más pesado.

## Por qué encaja con ABOS
El documento original describe "Autonomous PMO — gestiona presupuesto, dependencias, riesgos y recursos, detecta proyectos retrasados y sobrecostes" — esto es exactamente ese rol, hecho con CrewAI (framework de orquestación, no conflictúa con la orquestación propia de X1 porque correría como proceso externo, no dentro de la extensión).

## Bajado a Fase 2 (verificado 2026-07-04)
El propio repositorio se declara **"DRAFT stage"** — sin instalación documentada (ni `pip install`, ni variables de entorno, ni confirmación de si expone API REST o es solo un script). No es un candidato "listo para desplegar" como pensé al principio — hace falta abrir el código fuente directamente (no solo el README) para saber si es usable ya. [[Plexo]] pasa a ser el pick más seguro de esta categoría mientras tanto.
