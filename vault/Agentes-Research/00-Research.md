# Research / OSINT (rol ABOS: Research Agent)

- [[OpenOSINT]] — **elegido para Fase 1**. Servidor MCP propio, funciona con Claude/GPT-4/modelos locales.
- [[Taranis-AI]] — más orientado a recolección de noticias/inteligencia situacional continua.
- [[SpiderFoot]] — nuevo (Lote 1, 2026-07-04). 200+ módulos de automatización OSINT, el más maduro de los cuatro — para barridos amplios, no conversación dirigida.
- [[OSINT-Agent-Skills]] — nuevo (Lote 1, 2026-07-04). MCP puro con 23 herramientas (DNS, Shodan, certificados, brechas de datos, blockchain) — el de menor coste de integración.

Nota de uso: herramientas OSINT son legítimas para investigación de mercado/competencia (el caso de uso de X1), pero todas estas herramientas advierten explícitamente de uso autorizado solamente — no usar para vigilancia de personas sin consentimiento.
