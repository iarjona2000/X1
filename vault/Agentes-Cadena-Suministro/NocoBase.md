# NocoBase

**Qué hace:** plataforma no-code/low-code de código abierto para construir sistemas de gestión empresarial — incluye módulos ligeros de ERP: gestión de compras, inventario y pedidos. Pensado explícitamente como base sobre la que conectar agentes de IA a flujos de negocio reales, no solo como dashboard.

**Repositorio:** [github.com/nocobase/nocobase](https://github.com/nocobase/nocobase)

**Por qué es el primer candidato de este clúster nuevo:** cadena de suministro/procurement no estaba cubierto por ningún rol de los documentos ABOS originales ni por `agency-agents` de forma sólida — es un hueco genuino identificado en el plan de expansión (`06-Plan-de-Expansion-Masiva.md`, Lote 2). NocoBase se elige como primer candidato por ser una base madura y auto-alojable, no un proyecto de fin de semana — pendiente comparar con más candidatos especializados (ej. herramientas de negociación con proveedores, gestión de contratos de compra) en una pasada posterior.

**Nivel de integración:** 2 (self-hosted, API REST propia).

**Nota de honestidad:** en esta búsqueda inicial también aparecieron proyectos más pequeños/tempranos específicos de "agente de procurement" (negociación con proveedores, análisis de contratos) — no se documentan aquí todavía por no tener suficiente señal de madurez (actividad, adopción) para recomendarlos con confianza. Revisar en el siguiente lote de mantenimiento.
