# TradingAgents

**Qué hace:** Framework multi-agente open-source donde varios LLMs especializados (analistas, investigadores, traders, gestores de riesgo) **debaten entre sí** para simular decisiones de trading reales antes de ejecutarlas.

**Relevancia ABOS:** encaja con el módulo "Gestión de inversiones" y "Scenario Simulator" del documento original, no tanto con el CFO Agent genérico.
