# OmniParser

**Repo:** [microsoft/OmniParser](https://github.com/microsoft/OmniParser) — 24.999⭐, Jupyter Notebook, mantenido por Microsoft.

**Qué es:** herramienta de parseo de pantalla puramente basada en visión — localiza y clasifica elementos de UI a partir de una captura de pantalla, para que un agente controle cualquier interfaz gráfica sin depender del DOM o accesibilidad de la app.

**Por qué entra en CTO:** distinto de [[Browser-Use]] — Browser-Use opera sobre el DOM de páginas web (rápido, preciso, pero solo web); OmniParser funciona sobre cualquier interfaz gráfica (apps de escritorio, no solo navegador) a partir de píxeles, útil para automatización más allá del navegador.
