# Activepieces

**Repo:** [activepieces/activepieces](https://github.com/activepieces/activepieces) — 23.107⭐, TypeScript, activo.

**Qué es:** automatización de flujos con ~400 servidores MCP incorporados para conectar agentes de IA a herramientas — combina el modelo visual de n8n con soporte nativo de MCP desde el diseño.

**Por qué entra en COO:** alternativa open-source a n8n con integración MCP de fábrica, útil como referencia si algún flujo operativo necesita conectar directamente con el ecosistema MCP sin capas intermedias.

**Comparado con:** [[Kestra]] — Kestra prioriza robustez/escala para pipelines críticos; Activepieces prioriza velocidad de integración vía UI visual y MCP nativo, más parecido al caso de uso típico de n8n.
