# OSINT Agent Skills

**Qué hace:** servidor MCP con 23 herramientas OSINT en un único paquete Node.js sin dependencias — DNS, Shodan InternetDB, crt.sh (certificados TLS, útil para enumerar subdominios), Wayback CDX (archivo histórico de páginas), búsqueda de código en GitHub, OTX (AlienVault, inteligencia de amenazas), HIBP (Have I Been Pwned, brechas de datos), Etherscan (blockchain), y Mastodon.

**Repositorio:** listado en [soxoj/awesome-osint-mcp-servers](https://github.com/soxoj/awesome-osint-mcp-servers) (directorio curado de servidores MCP para OSINT — útil también como cantera para futuros lotes).

**Por qué se añade:** es Nivel 1 puro (MCP nativo, cero dependencias) — el coste de incorporación más bajo posible de los tres candidatos de este clúster. Cubre huecos muy específicos (certificados, blockchain, brechas de datos) que ni OpenOSINT ni SpiderFoot tienen como foco principal.
