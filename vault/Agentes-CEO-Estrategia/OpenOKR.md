# OpenOKR

**Qué hace:** Software de gestión de OKRs (Objectives and Key Results) open-source.

**Repositorio:** [github.com/openokr/openokr](https://github.com/openokr/openokr)
**Nota:** no es "IA" en sí — es la estructura de datos (objetivos/resultados clave) sobre la que X1 podría generar y actualizar OKRs usando su propia cascada NVIDIA+Gemini, en vez de necesitar IA propia del proyecto.
