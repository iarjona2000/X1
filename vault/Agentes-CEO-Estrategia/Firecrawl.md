# Firecrawl

**Repo:** [firecrawl/firecrawl](https://github.com/firecrawl/firecrawl) — 143.846⭐, TypeScript, activo, también disponible como API SaaS gestionada.

**Qué es:** API para buscar, scrapear e interactuar con la web a escala — convierte cualquier página o sitio entero en markdown/datos estructurados listos para que un LLM los procese, sin mantener infraestructura de scraping propia.

**Por qué entra en CEO-Estrategia:** complementa las herramientas OSINT ya catalogadas (que buscan personas/organizaciones/infraestructura) con una capacidad más general — convertir cualquier sitio web en material de research listo para análisis, útil para inteligencia de mercado y competencia.

**Comparado con:** [[OpenOSINT]] y [[SpiderFoot]] — esas herramientas están especializadas en inteligencia de fuentes abiertas (personas, dominios, infraestructura); Firecrawl es de propósito más general (cualquier sitio web a datos estructurados), sin el enfoque de seguridad/vigilancia de las herramientas OSINT.
